<h1>linux commands</h1>

<p>
    cd {./dir}                              change directory.
    cd ..                                   return to the parent directory.
    ls                                      show the files and directories in the current folder.
    ls {./dir}                              show the files and directories in respective.
    touch {name}                            creates a new file if it doesn't exists, else it updates the metadata of the file. Can create multiple files at once
    mkdir {name}                            creates a new directory.
    mv {origin (file || dir)} {target dir}  move files and directories to other directory.
</p>
